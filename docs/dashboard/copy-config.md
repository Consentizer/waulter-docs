# Copying Configurations

Waulter's configuration duplication feature lets you create copies of existing configurations — enabling a powerful template-based workflow for managing consent across multiple websites.

  - Create a "master" or "template" configuration with your standard settings

  - Keep the template configuration inactive — it's a blueprint, not a live config

  ## Enabling batch updates
  - When you need to update settings across multiple deployments:
    1. Update the template configuration with the new settings
    2. Duplicate the updated template for each affected client/site
    3. Review and activate the new copies
    4. Deactivate the old configurations
  - This workflow enables updates across 10, 50, or 100+ deployments in minutes
  - Especially powerful for agencies managing multiple client websites

  - Initial setup: create template → duplicate per client → customise per client → activate
