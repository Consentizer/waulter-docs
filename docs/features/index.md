# Features

Waulter provides a comprehensive set of consent management features. This section covers each feature in detail.

| Feature | Description | Guide |
|---------|-------------|-------|
| **Consent Banner** | A customisable, GDPR-compliant banner that collects visitor consent with Accept All, Reject All, and Customise options | [Consent Banner](consent-banner.md) |
| **Purposes** | Configurable cookie categories (analytics, marketing, personalisation) that map to Google Consent Mode signals | [Purposes](purposes.md) |
| **Google Consent Mode 2.0** | Automatic integration with GCM v2 — Basic and Advanced modes, all 7 consent types | [GCM 2.0](gcm2.md) |
| **Events & Data Layer** | Structured data layer events (`Waulter:Decision`, `Waulter:Loaded`, etc.) for GTM triggers and analytics | [Events](events.md) |
| **Policy Documents** | Render versioned Cookie Policy and Privacy Policy documents directly on your pages via `appendDocument` | [Documents](documents.md) |
| **User Sharing** | Cross-domain consent — visitors consent once and the decision is honoured across your domain portfolio | [User Sharing](user-sharing.md) |
| **Waulter Cookies** | How the SDK stores consent state using first-party cookies | [Cookies](cookies.md) |
| **Permission Transaction ID** | Tamper-evident consent receipts for GDPR accountability and audit trails | [Permission TX](permission-tx.md) |
